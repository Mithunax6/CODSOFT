# Task 1 - Number Guessing Game

This is a Java-based Number Guessing Game developed as part of the CodSoft Java Development Internship.

## 🎯 Features
- Random number generation (1 to 100)
- User input handling
- Feedback (Too High / Too Low / Correct)
- Limited number of attempts
- Multiple rounds option
- Score tracking

## ▶️ How to Run

1. Compile the program:
   javac NumberGuessingGame.java

2. Run the program:
   java NumberGuessingGame
