CodSoft Java Development Internship

This repository contains the tasks completed as part of the CodSoft Java Development Internship.

📌 Internship Overview
The internship focuses on building core Java programming skills through practical tasks and mini-projects using Object-Oriented Programming concepts.

✅ Task 1: Number Guessing Game
A simple Java game where the user tries to guess a randomly generated number.

Features:
- Random number generation
- User input handling
- Feedback (Too High / Too Low)
- Limited attempts
- Score tracking

✅ Task 2: Student Grade Calculator
A Java program that calculates total marks, average percentage, and assigns grades.

Features:
- Input marks for multiple subjects
- Calculates total and average
- Grade assignment based on percentage
- Input validation

🔄 Task 3: ATM Interface
A Java-based simulation of an ATM machine.

Functionalities:
- Withdraw money
- Deposit money
- Check account balance
- Input validation (sufficient balance)
- Menu-driven interface

🔄 Task 4: Currency Converter
A Java application to convert one currency to another.

Functionalities:
- Select base and target currency
- Convert amount between currencies
- Display converted result
- (Optional) Real-time exchange rates using API

🔄 Task 5: Student Management System
A Java application to manage student records using Object-Oriented Programming.

Functionalities:
- Add a new student
- Remove a student
- Search for a student
- Display all students
- Edit student details
- Input validation for correct data entry
- File handling (read/write student data)

Classes Used:
- `Student` (stores student details like name, roll number, grade)
- `StudentManagementSystem` (handles operations on students)

 🛠 Technologies Used
- Java
- OOP Concepts (Classes, Objects, Methods)
- File Handling
- Console-based User Interface
